## 🚀 Featured Projects

### 🚨 [Gaon Guard AI](https://github.com/Rahim36712/gao_ai)
An 11-agent agentic crisis-response system built for a Google Antigravity hackathon. Verifies disaster reports across 5 data sources, dispatches response teams using Haversine-based routing, and flags missing persons and aid fraud in real time.

<img src="https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white"/> <img src="https://img.shields.io/badge/Dart-0175C2?style=flat-square&logo=dart&logoColor=white"/> <img src="https://img.shields.io/badge/FastAPI-009688?style=flat-square&logo=fastapi&logoColor=white"/> <img src="https://img.shields.io/badge/Firebase-FFCA28?style=flat-square&logo=firebase&logoColor=black"/>

### 🧠 [Brain Tumor Segmentation (3D)](https://github.com/Rahim36712/Brain-Tumor-Segmentation-3D)
A 3D U-Net / Attention U-Net pipeline for volumetric MRI tumor segmentation on the BraTS dataset — full flow from preprocessing to training, evaluation (Dice, IoU, HD95), Grad-CAM explainability, and a Streamlit inference app.

<img src="https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white"/> <img src="https://img.shields.io/badge/PyTorch-EE4C2C?style=flat-square&logo=pytorch&logoColor=white"/> <img src="https://img.shields.io/badge/Streamlit-FF4B4B?style=flat-square&logo=streamlit&logoColor=white"/>

### 🚗 [Self-Driving Car Simulation](https://github.com/Rahim36712/self-driving)
Real-time lane detection (classical CV) and YOLOv8 object detection on dashcam video, fused by a rule-based decision engine. Multi-threaded pipeline sustaining 15+ FPS with a 6-panel debug view.

<img src="https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white"/> <img src="https://img.shields.io/badge/OpenCV-5C3EE8?style=flat-square&logo=opencv&logoColor=white"/> <img src="https://img.shields.io/badge/YOLOv8-111F68?style=flat-square&logo=ultralytics&logoColor=white"/>

### 📊 [GitHub Insights Dashboard](https://github.com/Rahim36712/github-dashboard)
A Streamlit analytics app that pulls the GitHub REST API for repo comparisons, language breakdowns, star growth trends, and contributor leaderboards.

<img src="https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white"/> <img src="https://img.shields.io/badge/Streamlit-FF4B4B?style=flat-square&logo=streamlit&logoColor=white"/> <img src="https://img.shields.io/badge/Plotly-3F4F75?style=flat-square&logo=plotly&logoColor=white"/>

### 🎵 [Automated Song Downloader](https://github.com/Rahim36712/N8N_SONG-_DOWNLOADING_AUTOMATION)
An n8n + Python pipeline that batch-downloads from YouTube/Spotify links or search queries, with concurrent threads, deduplication, CSV logging, and a Flask GUI.
